# Squad Shuffle

A Vite + React app for randomly splitting a list of names into balanced teams.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy on Vercel

Push this repository to GitHub, import it in Vercel, and use the default Vite settings:

- Build command: `npm run build`
- Output directory: `dist`
